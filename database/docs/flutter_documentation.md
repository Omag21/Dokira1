# Documentation Flutter – Projet Dokira

## 1. Objectif de l’application mobile
L’application mobile Dokira permet :
- la gestion des patients
- la réservation de consultations
- le suivi médical
- la messagerie patient – médecin
- la consultation du dossier médical

Elle communique avec le backend FastAPI via des endpoints sécurisés (JWT).

---

## 2. Architecture Flutter
L’architecture utilisée est **Clean Architecture + State Management Provider**.
